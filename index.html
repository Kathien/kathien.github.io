About Me

Projects

  • Virtual Front Panel

    • C++11
    • JSON
    • Qt
    • TCP/IP
    • MVVM

    Virtual front panel for a gating controller. To check and ensure proper functionality a small server imitating the controller itself was written. MVVM pattern and dependency injections were used.

  • Weather Radar

    • Android
    • REST
    • Material design
    • Google Maps v2 API
    • OpenWeatherMaps API

    This app was written to help people riding motorcycles plan their trips. App shows different weather conditions like precipitation, temperature, wind power worldwide. it focuses automatically on user’s location and allows to predict whether user will get into some storm or maybe just little drizzle.

  • DonorUa Application

    • Android
    • REST
    • Material design
    • Google Maps v2 API

    Application uses open API to show information about current patients who need blood, blood centers and their location on the map, news and events of the DonorUa organization.
